# Groove AI Platform

AI-powered Meta campaign analytics dashboard built using React, TypeScript, Express, Prisma, and Supabase.

## Features

- Campaign analytics dashboard
- AI-generated campaign insights
- AI campaign assistant
- Prisma ORM integration
- Supabase PostgreSQL database
- Recharts analytics visualizations

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Recharts
- Vite

### Backend
- Node.js
- Express
- Prisma ORM
- Supabase PostgreSQL

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```
